# UiTM-Students-Withdrawal-System
The UiTM Students Withdrawal System is a platform designed to streamline and manage the process of student withdrawals at UiTM. The system allows students to submit their withdrawal requests, track the status of their applications, and provides administrative staff with tools to efficiently process these requests.

FIRSTLY, after you download the zip files, you must extract the files into xampp/htdocs folder.

MAKE SURE THE FILE NAME IS: UiTM-Students-Withdrawal-System-main

SECONDLY, open phpmyadmin and create a new database named "ims566_db".

THIRDLY, import the db.sql file into the database.


TO NAVIGATE THE SYSTEM:
User must use http://localhost/UiTM-Students-Withdrawal-System-main/login.php for the login page.
